# Spambot

Spam bot for 6b6t.org using Mineflayer


## Config Format

```json
{
  "username": "BotName",
  "password": "BotPassword",
  "messages": [
    "message1", 
    "message2",
  ],
  "blacklist": ["User1", "User2"]
}
```


## Deployment

To run the bot you need npm and nodejs

First install mineflayer module

```bash
  npm install mineflayer
```

Run the bot
```bash
  node app.js
```
